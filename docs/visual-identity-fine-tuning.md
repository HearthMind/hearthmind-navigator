# Navigator Fine-Tuning — Visual Identity & Trust Cues

**Purpose:**
Ensure Navigator's visual identity reinforces safety, legitimacy, calm, and emotional accessibility across all interface scales.

---

## 1. Core Brand Assets

Navigator uses HearthMind branding as a trust anchor, not decoration.

### Required Assets:

**✅ Master HearthMind Logo (Transparent)**
- Full flame + "HearthMind Systems"
- Used for:
  - Landing hero
  - Grant decks
  - Press materials

**✅ Small UI Flame Mark**
- Flame icon only
- Used for:
  - Header/nav
  - Section dividers
  - Loading states

**✅ Favicon / App Icon Set**
- Tiny flame only
- Used for:
  - Browser tab
  - PWA / desktop wrapper

**Monochrome Variants Required:**
- White on transparent
- Black on transparent
- (For print, watermarks, and accessibility contrast modes)

---

## 2. Emotional Function of Visual Design

Navigator's UI must:
- Reduce intimidation
- Avoid bureaucratic harshness
- Signal:
  - "You're safe here"
  - "You're not being judged"
  - "This system helps without shaming"

This is why:
- The flame is small and steady, not aggressive
- The gold is warm, not clinical yellow
- The navy is calm, not sterile black

---

## 3. Micro-Copy + Visual Pairing Rule

Each major UI card pairs:
- ✅ A functional label
- ✅ A reassurance line
- ✅ A visual anchor (consistent icon rhythm)

**This is now part of Navigator's accessibility doctrine, not optional polish.**

---

## 4. Technical Constraint (for Demo & Grants)

All logo assets must be available as:
- SVG (scalable)
- PNG (transparent)
- Optimized for dark UI

No raster-only branding allowed in the final demo build.

---

## 5. Logo Asset Set Specification

### Master Transparent Logo
- Full flame + "HearthMind Systems"
- Transparent background (PNG + SVG)
- Sizes: 2048×2048 (master), 1024×1024, 512×512

### Small Header Logo
- Flame icon only
- No text
- Optimized for dark UI
- High-contrast edges
- Sizes: 128×128, 64×64, 32×32

### App Icon / Favicon Set
- Flame only, centered
- Thickened lines for legibility at tiny sizes
- Sizes: 48×48, 32×32, 16×16

### Asset Naming Convention
- `hearthmind-logo-master.svg`
- `hearthmind-logo-master-2048.png`
- `hearthmind-flame-ui-128.png`
- `hearthmind-flame-ui-64.png`
- `hearthmind-flame-ui-32.png`
- `hearthmind-favicon-48.png`
- `hearthmind-favicon-32.png`
- `hearthmind-favicon-16.png`
- `hearthmind-logo-mono-white.svg`
- `hearthmind-logo-mono-black.svg`
